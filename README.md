# chef-simple-dcos

Simple cookbook to test DCOS. It was created for a DCOS POC.

It's a wrapper to the community cookbook.

## Supported Platforms

* CentOS 7

## Usage

Include `chef-simple-dcos::master` or `chef-simple-dcos::slave` in your
node's `run_list`.
